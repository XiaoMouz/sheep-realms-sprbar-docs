# 命令使用手册

本手册详细描述了各个命令的作用

## 命令列表

### baidu

直接在百度搜索关键词

eg:

```powershell
/baidu MCBBS  //百度一下mcbbs
```



### baidu-img

使用百度搜图搜索关键词

eg:

```powershell
/baidu-img 色图  //百度搜图搜索色图
```



### bilibili

在bilibili搜索关键词

eg:

```powershell
/bilibili 114514  //在bilibili搜索大粪
```



### bilibili-aid

直接访问bilibili里该av号视频

eg:

```powershell
/bilibili-aid av12755553  //直接访问av号为av12755553的视频
```



### bilibili-au

直接访问bilibili里该au号音频

eg

```powershell
/bilibili-au au?  //访问au号为？音频，用问号是因为找不到啥好活，实际使用请接数字
```



### bilibili-bvid

直接访问bilibili里该bv号视频

eg:

```powershell
/bilibili-bvid BV1Ux411W79m  //访问bv号为BV1Ux411W79m的视频
```



### bilibili-cv

直接访问bilibili里该cv号的文章

eg:

```powershell
/bilibili-cv cv911  //访问cv号为911的文章（随意打的）
```



### bilibili-ss

直接访问bilibili同号番剧

eg:

```powershell
/bilibili-ss ss36175 //访问ss号为36175的番剧
```

ss号是播放番剧时上方url末尾数字

### bilibili-uid

直接访问该uid用户

eg:

```powershell
/bilibili-uid 208259  //访问uid为208259的用户
```



### bing

bing搜索，与baidu搜索同理（bing更干净，大嘘）

eg:

```powershell
/bing mcbbs  //bing搜索mcbbs
```



### enwiki

在英文维基搜索关键词

eg:

```powershell
/enwiki minecraft  //在英文维基查找minecraft
```



### google

谷歌一下（由于受到国内防火墙限制，如需正常使用请自备科学上网方法）

eg:

```powershell
/google keyword
```



### google-img

谷歌搜图（依旧受到国内防火墙限制，请自备科学上网方法）

eg:

```powershell
/google-img pornhub
```



### jawiki

在日本维基搜索页面（与enwiki同理）

eg:

```
/jawiki minecraft  //在日文维基查找minecraft
```



### log

向开发者控制台发送字符

eg:

```powershell
/log Helloworld
```

然后按F12打开开发者控制台，查看Console（控制台）窗口

### mcbbs

在mcbbs搜索此关键词

eg:

```powershell
/mcbbs minecraft  //在mcbbs搜索minecraft关键词
```



### mcbbs-aid

直接访问id为“你填写的id”的文章

eg:

```powershell
/mcbbs-aid anyaid  //实际一般不使用，因为根本8知道文章id是多少
```



### mcbbs-fid

访问id为“你填写id”的版块

eg:

```powershell
/mcbbs-fid news  //访问新闻版块
```



### mcbbs-pid

查看id为“你填写id”的帖子

eg:

```powershell
/mcbbs-pid 114514  //访问大粪贴
```



### mcbbs-tagid

查看id为"你填写id"的标签

eg:

```powershell
/mcbbs-tagid 2946  //访问id为2946的标签页，查看标签带“求助贴”的帖子
```

### mcbbs-tagname

搜索该标签

eg:

```powershell
/mcbbs-tagname 求助帖
```



### mcbbs-tid

查看id为“你填写id”的主题

eg:

```powershell
/mcbbs-tid 7808  //查看id为7808的主题
```



### mcbbs-uid

查看uid为“你填写id”的用户页

eg:

```powershell
/mcbbs-uid 2153967  //查看uid为2153967的用户页
```



### mcbbs-username

查看用户名为“你填写用户名”的用户页

eg:

```powershell
/mcbbs-uesrname  //查看你自己的用户页（没错，就是一个空格）
```



### mcbbs-user-search

搜索一个用户

eg:

```powershell
/mcbbs-user-search admin  //搜索用户名为admin的用户（包括包含）
```



### mcbbs-user-space

访问为”你填写uid“用户的用户页（其实和mcbbs-uid`冲突了？）

eg:

```powershell
/mcbbs-user-space 1262947  //查看uid为1262947的用户页
```



### mediawiki

在mediawiki搜索wiki

eg:

```powershell
/mediawiki minecraft  //搜索带有minecraft关键词的wiki
```



### moegirl

在萌娘百科搜索关键词

eg:

```powershell
/moegirl shota  //在萌娘百科搜索shota
```



### sogou

搜狗搜索（与其他引擎一致，个人感觉体验一般）

### touhouwiki

在东方（东方project）wiki搜索关键词

eg:

```powershell
/touhouwiki Tobi魔理沙  //在东方wiki搜索Tobi魔理沙
```



### zhwiki

在中文wiki搜索关键词

eg:

```powershell
/zhwiki minecraft  //在中文维基查找minecraft
```
